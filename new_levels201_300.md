# New Level Design: Levels 201-300

This design adds 100 new levels without changing the existing first 200 levels.
It introduces higher-level grammar, richer functional English, and new game formats.

## Suggested New Game Formats

These fresh formats can be introduced in levels 201-300 without affecting the first 200 levels.

- `sentence_scramble`: reorder word tiles to build a grammatically correct sentence.
- `dialogue_fill`: choose the correct phrase to complete a short conversation.
- `tense_turn`: select the correct verb form for a sentence in the target tense.
- `article_hunt`: tap the correct article or determiner for each noun phrase.
- `modal_mission`: choose the best modal verb for a real-world sentence.
- `condition_match`: complete conditional sentences with the right verb tense.
- `voice_switch`: convert active sentences into passive or vice versa.
- `pronoun_puzzle`: replace repeated nouns with the correct pronoun.
- `phrase_pair`: match questions to their best responses.
- `preposition_path`: place objects into boxes labeled by location or movement.

## Level Design Notes

- Each level uses one main new topic but may include supporting vocabulary from related areas.
- New worlds are sequenced from grammar fundamentals to functional communication and then to everyday vocabulary.
- Existing game formats can be reused where they fit naturally; new formats are reserved for the new levels.
- The first 200 levels remain unchanged; these 100 new levels are designed as a second stage of the curriculum.
